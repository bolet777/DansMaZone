Contribuer à DansMaZone

Vous pouvez aider à améliorer cette extension en contribuant de nouveaux sites pour toute la communauté!

Via GitHub

Vous pouvez contribuer directement au code source en proposant des modifications sur GitHub:

  1. Visitez notre dépôt GitHub
  2. Modifiez le fichier category-sites.js
  3. Créez une pull request avec vos ajouts
Voir sur GitHub

Par email

Vous pouvez également nous envoyer vos suggestions par email:

  1. Exportez vos sites personnalisés (bouton "Exporter mes sites")
  2. Envoyez le fichier à ccosenza.dlab@gmail.com
  3. Nous les intégrerons dans la prochaine version
